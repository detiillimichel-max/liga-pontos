🔵 Liga Pontos
Puzzle de conexão de pontos coloridos — jogue no navegador, funciona offline
🎮 Como Jogar
Conecte os pares de pontos da mesma cor
Preencha todas as células do grid
Não deixe nenhuma célula vazia para vencer
Quanto menos jogadas usar, mais estrelas você ganha!
Toque e arraste para traçar o caminho entre os pontos.
✨ Funcionalidades
🎨 60 níveis em 4 dificuldades (Fácil, Médio, Difícil, Expert)
📱 PWA — instale no celular como app nativo
💾 Progresso salvo automaticamente (localStorage)
⭐ Sistema de estrelas (1–3 por nível)
💡 Sistema de dicas (3 por nível)
🎉 Animação de vitória com partículas
📴 Funciona offline (Service Worker)
🌙 Dark mode com glassmorphism
